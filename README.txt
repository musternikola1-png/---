MedHub Telegram Mini App

Как загрузить на Netlify:
1. Зайди на netlify.com
2. Add new site -> Deploy manually
3. Перетащи папку medhub-mini или архив medhub-mini.zip
4. Получи ссылку вида https://...netlify.app
5. Эту ссылку вставь в BotFather как Web App URL или в кнопку бота.

Файл запуска: index.html
